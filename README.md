# CCTradeWelcomeBot

## Setup
1. Create a public Telegram channel.
2. Add the bot as administrator.
3. Set environment variable TOKEN.
4. Install dependencies:
   pip install -r requirements.txt
5. Run:
   python bot.py

IMPORTANT:
Replace CHANNEL in bot.py with your public channel username, e.g. @CCTRADE.
Force Join cannot verify membership using a private invite link.
